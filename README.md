# Tronix
Tronix is a futuristic TRON-inspired lightcycle racing game built with HTML, CSS, and JavaScript, featuring neon visuals, boost mechanics, and interactive controls in a glowing grid arena.

#Controls:
    
W → Forward
S → Backward
A → Left
D → Right
Shift → Boost
